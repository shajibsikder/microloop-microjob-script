# MicroLoop – Microjob & Microtask Marketplace Script

MicroLoop is a PHP-based microjob and microtask marketplace platform where users can post jobs and complete tasks to earn money.

This project was developed by **Md Shajib Sikder**.

## Features

- User Dashboard
- Post New Job
- Find Job
- Task Proof System
- Wallet System
- Deposit & Withdraw
- Spin & Earn
- Investment Plans
- Referral System
- Advertisement System
- Support Ticket System
- Admin Control Panel

## User System

Users can work in two roles:

- Job Poster (Employer)
- Job Worker (Task Completer)

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
